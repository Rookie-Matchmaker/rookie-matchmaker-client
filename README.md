A bot that will act as a match maker using ingame lobbies. Focused at new players.
Find live version on [here](http://lobbyfinder.devinda.me/)

## Installation

```bash
npm install
```

## Usage in Dev Server

```bash
npm run dev
```

## Usage in Prod

```bash
npm run build
```
